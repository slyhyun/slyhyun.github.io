Numpy와 Scikit-learn을 사용하여 MNIST 데이터셋을 기반으로 다중 클래스 분류를 위한 로지스틱 회귀 모델을 구현하고 학습하는 방법을 설명하는 프로젝트

<!--more-->
```
1.  개요
이 보고서는 Numpy와 Scikit-learn을 사용하여 MNIST 데이터셋을 기반으로 다중 클래스 분류를 위한 로지스틱 회귀 모델을 구현하고 학습하는 방법을 설명합니다. 모델 학습에 필요한 데이터 전처리, 학습 및 평가 방법을 다루며, 사용자가 조정할 수 있는 학습 파라미터를 포함합니다.

2. 구현 방법
- 라이브러리 임포트 : numpy, sklearn.datasets, sklearn.model_selection, sklearn.preprocessing, matplotlib.pyplot를 임포트하여 데이터 로드, 전처리, 모델 구축 및 평가, 시각화를 수행합니다.
- Argument Parser 설정 : argparse를 사용하여 사용자 정의 옵션을 설정합니다. 예를 들어, 미니배치 크기와 최대 에폭 수를 사용자 정의할 수 있습니다.
- MNIST 데이터셋 로드 및 전처리 : datasets.fetch_openml을 사용하여 MNIST 데이터셋을 로드합니다. 데이터를 정규화하고, 각 픽셀 값의 평균을 뺍니다. 레이블 데이터를 원-핫 인코딩하여 다중 클래스 분류에 사용할 수 있도록 변환합니다.
- 데이터 분할 : train_test_split을 사용하여 데이터를 학습, 검증, 테스트 세트로 분할합니다. 먼저 전체 데이터를 학습 세트와 임시 세트로 분할한 후, 임시 세트를 다시 검증 세트와 테스트 세트로 분할합니다.
- 모델 생성 및 학습 : LogisticRegression 클래스를 인스턴스화하여 모델을 생성합니다. 학습 데이터를 사용하여 모델을 학습시키고, 검증 데이터를 사용하여 모델 성능을 평가합니다.
- 모델 평가 : 테스트 데이터셋을 사용하여 학습된 모델의 최종 성능을 평가합니다. 모델의 테스트 정확도를 출력합니다.
- 결과 시각화 : matplotlib.pyplot을 사용하여 첫 번째 이미지 데이터를 시각화합니다. 이는 데이터가 올바르게 로드되고 전처리되었는지 확인하는 데 유용합니다.

3. 주요 구현 코드 요약
- Argument Parser 설정 : 사용자로부터 미니배치 크기와 최대 에폭 수를 입력받아 설정합니다.
- 데이터 로드 및 전처리 : MNIST 데이터셋을 로드하여 정규화 및 원-핫 인코딩을 수행합니다. 데이터를 학습, 검증, 테스트 세트로 분할합니다.
- 모델 학습 : LogisticRegression 클래스를 사용하여 모델을 학습시킵니다. 미니배치 경사하강법과 Early Stopping을 사용하여 모델의 학습 과정을 최적화합니다.
- 모델 평가 : 테스트 세트를 사용하여 모델의 성능을 평가하고, 정확도를 출력합니다.
- 결과 시각화 : 학습된 첫 번째 이미지를 시각화하여 데이터가 올바르게 처리되었는지 확인합니다.

4. 결론
이 방법을 통해 다중 클래스 분류 문제를 해결하기 위한 로지스틱 회귀 모델을 효과적으로 구축하고 학습할 수 있습니다. 사용자가 학습 파라미터를 조정할 수 있도록 설정하여 모델의 유연성을 높였으며, Early Stopping 기법을 적용하여 과적합을 방지하고 모델의 일반화 성능을 향상시켰습니다. 데이터 전처리와 분할 과정을 통해 데이터의 품질을 보장하고, 최종 모델의 성능을 정확하게 평가할 수 있습니다.
```
