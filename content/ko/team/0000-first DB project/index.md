시네마 리뷰 - 영화의 정보를 제공하고, 리뷰를 통하여 영화를 평가할 수 있는 웹 페이지를 제작하는 프로젝트

<!--more-->
```
1. 구현 기능
- 회원가입, 로그인 및 로그아웃
- 회원 목록, 정보 열람 및 관리
- 영화 검색 기능
- 영화 정보 생성, 읽기, 수정, 삭제 기능
- 댓글(리뷰) 생성, 읽기, 수정, 삭제 기능
- 북마크 생성, 삭제 기능

2. 개발 언어
- BACK-END : Java(서블릿)
- FRONT-END : JavaScript + CSS + JSP
- DATABASE : MySQL
```
